Mot à trouver:

Vous devez trouver un mot du dictionnaire. Pour ce faire il vous suffit de cliquer sur une des lettres du clavier, si cette lettre est dans le mot elle apparaitra à sa place, sinon le dessin du "pendu" va se compléter. Le jeu se termine quand le dessin est complet ou quand vous avez trouvé le mot. La première et la dernière lettre sont offerte, ces deux lettres peuvent être présentes dans le mot.
